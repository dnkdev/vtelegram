# VTelegram

Telegram Bot API wrapper for V language. v 0.1.0
<br>
All methods are in lowercase instead of snakeCase: `getupdates` instead of `getUpdates`.
<br>
It has Telegram Bot API 6.4 models.
Already could be used at this moment.
